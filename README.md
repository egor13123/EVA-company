# EVA TSK — Static Export

Готовая статическая версия сайта eva-tsk.ru для загрузки на любой обычный хостинг (например, Sprinthost).

## Что внутри

Полностью собранный сайт: HTML + CSS + JS + изображения. Никаких Node.js, баз данных, серверной логики не требуется.

## Как использовать

1. На этой странице GitHub: **Code → Download ZIP**
2. Распакуйте архив
3. Содержимое (`index.html`, `_next/`, `about/`, `services/` и т.д.) загрузите в корневую папку сайта на хостинге (обычно `public_html`)
4. Привяжите домен, включите SSL — готово

## Источник

Исходный код приложения — в основной ветке `main`. Эта ветка обновляется отдельно после каждой пересборки.

Сборка: `npm run build` в основном проекте → папка `out/` копируется сюда.
